# Product Page (Vite + React)

Simple product listing and detail app built with Vite and React.

Features:
- Product Listing page fetching from https://dummyjson.com/products
- Client-side pagination (4 products per page)
- Clickable cards navigate to Product Detail page
- Product Detail fetches details by ID

Run (PowerShell):

```powershell
# install dependencies
npm install
# dev server
npm run dev
# build
npm run build
# preview build
npm run preview
```

Notes:
- Uses only built-in fetch API and pure CSS. No external UI libraries.
